Java İçerikleri

Başlangıç Seviyesi Java

Bu bölümde Java'nın temel kavramlarını ve syntax'ını öğreneceksiniz. Değişkenler, döngüler, koşullar, fonksiyonlar gibi temel yapıları ele alacağız. Örneklerle destekleyerek konuları daha iyi anlayacaksınız. 


        public class HelloWorld {
          public static void main(String[] args) {
            String message = "Merhaba, dünya!";
            System.out.println(message);
          }
        }

Nesne Yönelimli Programlama (OOP)

Java'nın nesne yönelimli programlama prensiplerini bu bölümde inceleyeceğiz. Sınıflar, nesneler, kalıtım, çok biçimlilik gibi OOP kavramlarını ele alarak nesne yönelimli programlamanın avantajlarını vurgulayacağız. 


        public class Person {
          private String name;
          private int age;

          public Person(String name, int age) {
            this.name = name;
            this.age = age;
          }

          public void sayHello() {
            System.out.println("Merhaba, ben " + name + ". " + age + " yaşındayım.");
          }
        }

Java API'leri

Bu bölümde Java'nın standart kütüphanelerinden (Java API'leri) bahsedeceğiz. String işlemleri, koleksiyonlar (List, Set, Map), dosya işlemleri, veritabanı bağlantısı gibi konuları ele alarak bu API'leri nasıl kullanabileceğinizi öğreneceksiniz. 


        import java.util.ArrayList;

        public class ArrayListExample {
          public static void main(String[] args) {
            ArrayList names = new ArrayList<>();
            names.add("Ali");
            names.add("Ayşe");
            names.add("Mehmet");

            for (String name : names) {
              System.out.println(name);
            }
          }
        }

GUI Uygulama Geliştirme

Java Swing veya JavaFX gibi GUI (Grafiksel Kullanıcı Arayüzü) kütüphanelerini kullanarak masaüstü uygulamaları geliştirmeyi bu bölümde ele alacağız. Formlar, düğmeler, etkileşimli bileşenler gibi GUI elemanlarını oluşturmayı ve kullanmayı öğreneceksiniz. 


        import javax.swing.*;

        public class HelloWorldGUI {
          public static void main(String[] args) {
            JFrame frame = new JFrame("Merhaba, Dünya!");
            frame.setDefaultCloseOperation(JFrame.EXIT_ON_CLOSE);

            JLabel label = new JLabel("Merhaba, dünya!");
            frame.getContentPane().add(label);

            frame.pack();
            frame.setVisible(true);
          }
        }

Web Geliştirme

Bu bölümde Java EE (Enterprise Edition) veya Spring Framework gibi Java tabanlı web geliştirme teknolojilerini ele alacağız. Servletler, JSP (JavaServer Pages), MVC (Model-View-Controller) mimarisi gibi konulara odaklanarak web uygulamaları geliştirmek için bu teknolojileri nasıl kullanacağınızı öğreneceksiniz. 


        import javax.servlet.*;
        import javax.servlet.http.*;
        import java.io.IOException;

        public class HelloWorldServlet extends HttpServlet {
          public void doGet(HttpServletRequest request, HttpServletResponse response) throws ServletException, IOException {
            response.setContentType("text/html");
            PrintWriter out = response.getWriter();
            out.println("");
            out.println("");
            out.println("
Merhaba, Dünya!
");
            out.println("");
            out.println("");
          }
        }

Veritabanı Erişimi

Bu bölümde Java'da veritabanıyla nasıl etkileşim kurabileceğinizi öğreneceksiniz. JDBC (Java Database Connectivity) kullanarak veritabanına bağlanma, sorguları çalıştırma ve sonuçları işleme gibi konuları ele alacağız. 


        import java.sql.*;

        public class DatabaseExample {
          public static void main(String[] args) {
            try {
              Connection connection = DriverManager.getConnection("jdbc:mysql://localhost:3306/mydatabase", "username", "password");

              Statement statement = connection.createStatement();
              ResultSet resultSet = statement.executeQuery("SELECT * FROM customers");

              while (resultSet.next()) {
                String name = resultSet.getString("name");
                int age = resultSet.getInt("age");
                System.out.println("Name: " + name + ", Age: " + age);
              }

              resultSet.close();
              statement.close();
              connection.close();
            } catch (SQLException e) {
              e.printStackTrace();
            }
          }
        }
