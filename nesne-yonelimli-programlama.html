Nesne Yönelimli Programlama (OOP)

Nesne yönelimli programlama (OOP), programlama paradigmasının bir türüdür. OOP, programları gerçek dünya nesneleriyle ilişkilendirerek düzenlemeye odaklanır. Programı, nesnelere ve nesneler arasındaki ilişkilere dayandırır. Her nesne, verileri (özellikler) ve bu veriler üzerinde işlem yapabilen fonksiyonları (metodlar) içerir. OOP'nin temel prensipleri arasında kalıtım, çok biçimlilik, soyutlama ve kapsülleme bulunur. Kalıtım, bir sınıfın başka bir sınıftan özelliklerini ve davranışlarını devralmasını sağlar. Çok biçimlilik, farklı nesnelerin aynı isimle farklı davranabilmesini sağlar. Soyutlama, detayları gizleyerek sadece gerekli olanlara erişimi sağlar. Kapsülleme, verilerin ve metodların bir araya getirilerek dış etkilere karşı korunmasını sağlar. OOP, yazılımın modülerliğini artırır, tekrar kullanılabilirliği sağlar ve büyük projelerin daha iyi yönetilmesini sağlar. Nesne yönelimli programlama, birçok programlama dilinde kullanılır ve Java gibi dillerde özellikle yaygındır. 


        public class Person {
          private String name;
          private int age;

          public Person(String name, int age) {
            this.name = name;
            this.age = age;
          }

          public void sayHello() {
            System.out.println("Merhaba, ben " + name + ". " + age + " yaşındayım.");
          }
        }
